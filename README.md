# Student-Performance-Analysis-Excel
Student Performance Analysis Using MS Excel
Objective

The main objective of this project is to study and analyze the academic performance of students based on factors such as attendance percentage, study hours, gender, and subject-wise marks using Microsoft Excel.

Tools Used

1)Microsoft Excel

2)Pivot Tables

3)Charts and Graphs

Analysis Performed

Classification of students into different performance levels based on average marks

Analysis of average marks of students according to gender

Comparison of subject-wise average marks

Analysis of the relationship between attendance and student performance

Conclusion

From the analysis, it can be concluded that student performance is influenced by factors such as attendance and study hours. The project shows that Microsoft Excel is an effective tool for performing basic data analytics, data visualization, and drawing meaningful insights from educational data.
The student performance analysis project successfully demonstrated how Microsoft Excel can be used as a powerful tool for basic data analytics. By using pivot tables and charts, meaningful insights were derived from student data. This project highlights the importance of attendance and study habits in improving academic performance and shows the practical application of Excel in data analysis.
